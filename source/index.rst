Football_injury_prediction documentation
=======================================================================

Cette documentation décrit le processus de développement d’un système de
prédiction des blessures et de recommandations personnalisées pour les joueurs
de football. Vous y trouverez une description des modèles utilisés, des
méthodes de prétraitement des données, ainsi que des évaluations des performances.

Cette documentation couvre les aspects suivants du projet :
   1. Introduction
   2. Prétraitement des données
   3. Entraînement des modèles
   4. Évaluation des performances
